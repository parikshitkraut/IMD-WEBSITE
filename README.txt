===========================================================
      IMD NAGPUR REGIONAL METEOROLOGICAL CENTRE
                  WEATHER DESK APP
===========================================================

This pendrive contains the complete web application for the 
IMD Nagpur Regional Weather Observations & Analytics. 

It can be run locally on ANY computer (Windows or macOS/Linux) 
without needing any internet connection, as it includes local 
database files and offline fallbacks!

-----------------------------------------------------------
PREREQUISITES:
-----------------------------------------------------------
To run this application, the computer needs to have Node.js 
installed. 

If Node.js is not installed:
1. Go to: https://nodejs.org/
2. Download and install the "LTS" (Recommended) version.
3. It takes less than 2 minutes to install.

-----------------------------------------------------------
HOW TO RUN THE SITE:
-----------------------------------------------------------

■ FOR WINDOWS USERS:
  1. Open the project folder on the pendrive.
  2. Double-click the "run.bat" file.
  3. Two terminal windows will open to start the servers.
  4. Your web browser will automatically open: http://localhost:5173

■ FOR MAC & LINUX USERS:
  1. Open the project folder on the pendrive.
  2. Double-click the "run.sh" file (or open Terminal, drag-and-drop the file, and press Enter).
  3. Your web browser will automatically open: http://localhost:5173

-----------------------------------------------------------
TO STOP THE APP:
-----------------------------------------------------------
Simply close the terminal windows that were opened by the launcher.
